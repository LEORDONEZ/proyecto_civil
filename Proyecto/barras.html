Cálculo de Acero en Vigas

Descripción General

Este programa está diseñado para ayudar en la selección de la combinación adecuada de barras de acero para vigas, cumpliendo con la norma NSR-10. El usuario puede ingresar el área de acero requerida, y el programa calculará la combinación óptima de barras, asegurando que se cumpla la base mínima requerida para la viga. Además, se ofrece una tabla con datos detallados de las barras disponibles, incluyendo diámetro, área y peso.

Funcionalidades Clave

  • Ingreso de Área de Acero Requerida:

    • El usuario introduce el área de acero necesario para la viga.
    • El programa utiliza esta información para calcular la combinación adecuada de barras.

  • Cálculo de Combinaciones de Barras:

    • Basado en el área de acero requerido, el programa determina la mejor combinación de barras de acero disponibles.
    • Se asegura de que la combinación elegida cumpla con la base requerida de la viga según la norma NSR-10.

  • Verificación de Cumplimiento de Base:

    • El programa verifica que la combinación de barras seleccionada cumpla con la base mínima necesaria para la viga, según las especificaciones de la norma NSR-10.

  • Tabla de Datos de Barras:

    • Una imagen detallada muestra todas las barras de acero disponibles, incluyendo:

      • Diámetro: El diámetro de cada tipo de barra.
      • Área: El área de acero de cada barra.
      • Peso: El peso de cada barra.
